# Contributing to QR Menu

First off, thank you for considering contributing to QR Menu! It's people like you that make it such a great tool.

## Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our [Issues](https://github.com/Softenrj/qr-menu/issues) section first to see if someone else has already created it. 
If not, feel free to **open a new issue**!

## Getting Started for Contributors

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally:
   ```bash
   git clone https://github.com/Softenrj/qr-menu.git
   ```
3. **Install the dependencies**:
   ```bash
   npm install
   ```
4. **Set up environment variables:** Check the `README.md` for required `.env` configurations.
5. Create a **new branch** for your feature or bug fix:
   ```bash
   git checkout -b feature/your-feature-name
   # OR
   git checkout -b fix/your-bug-fix
   ```

## Making Changes

- Ensure your code follows the existing style, which relies heavily on **TypeScript**, **Next.js 15**, and **Tailwind CSS**.
- Run the development server to preview your changes:
  ```bash
  npm run dev
  ```
- Before pushing, make sure your code builds successfully and has no linting errors:
  ```bash
  npm run build
  npm run lint
  ```

## Submitting a Pull Request

1. Commit your changes with a descriptive, concise commit message.
   ```bash
   git commit -m "feat: descriptive message of your addition"
   ```
2. Push your branch to your fork:
   ```bash
   git push origin feature/your-feature-name
   ```
3. Open a **Pull Request (PR)** on the main repository. 
4. Include a clear description of the problem you are solving, the changes you made, and any relevant issue numbers.
5. Wait for a maintainer to review your PR. We will try to review as soon as possible!

## Any questions?

If you run into any trouble, have any questions, or would just like some guidance, don't hesitate to reach out! You can open a discussion on GitHub or email [rjsharmase@gmail.com](mailto:rjsharmase@gmail.com).

**Thank you for your contributions! 🚀**
